Ласкаво просимо у світ Git

Git запам'ятовує всі версії вашого коду. Це як «машина часу»: ви можете сміливо пробувати нові ідеї, а потім у будь‑який момент повернутися до стабільного стану проєкту.

1. Почитайте теорію

Зайдіть у розділ «Про Git». Там коротко пояснено, хто створив Git, навіщо він був потрібен та які головні ідеї стоять за ним.

2. Подивіться приклади

У розділах «Репозиторії та гілки (основи)» і «Приклади команд Git» ви знайдете найпростіші команди: git init, git add, git commit, роботу з гілками.

3. Закріпіть тестом

Відкрийте розділ «Тест» і перевірте себе. Якщо якась відповідь незрозуміла — поверніться до розділу з теорією і перечитайте його.

Візуально: як виглядає Git

На схемі нижче показано просту модель: гілка main, кілька комітів та окрема гілка для експериментів.
