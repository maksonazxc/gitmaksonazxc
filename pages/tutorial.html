Початок роботи з Git

  • Встановіть Git з офіційного сайту для вашої ОС.
  • Відкрийте термінал і задайте ім'я та email, які будуть видно в історії комітів:
    git config --global user.name "Ваше ім'я"
    git config --global user.email "you@example.com"
  • Створіть папку проєкту, зайдіть у неї і виконайте git init — так ви "вмикаєте" Git у цій папці.
  • Створіть файл (наприклад, readme.txt), потім збережіть першу версію:
    git add .
    git commit -m "Перший коміт"
  • Далі можна змінювати файли і робити нові коміти. Не бійтеся — Git дозволяє повернутися назад.
