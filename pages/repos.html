Що таке репозиторій і гілка?

Репозиторій — це папка з вашим проєктом, у якій Git зберігає історію змін. Ви працюєте з файлами як завжди, але Git пам'ятає, що і коли змінилось.

Гілка — це окрема «лінія» розвитку проєкту. У ній можна пробувати нові ідеї, не чіпаючи основну гілку main.

  1. Створіть папку і зайдіть у неї, потім: git init — створення репозиторію.
  2. Додайте файли і зробіть перший коміт: git add ., потім git commit -m "Перший коміт".
  3. Створіть гілку для експериментів: git switch -c feature-idea.
  4. Повернутися в основну гілку можна командою git switch main.
