Git + GitHub: у чому різниця?

Git — це локальна система контролю версій. Вона встановлюється на твій комп'ютер і дозволяє створювати репозиторії, робити коміти, працювати в гілках і повертатися до будь-якої версії коду.

GitHub — це веб-сервіс, який зберігає репозиторії Git у хмарі. Він дає змогу:

  • ділитися кодом з іншими;
  • працювати в команді через pull request;
  • створювати issues та обговорювати задачі;
  • налаштовувати автоматизацію через GitHub Actions.

Коротко: Git = інструмент на твоєму ПК, а GitHub = місце, де твої Git‑репозиторії живуть онлайн.
